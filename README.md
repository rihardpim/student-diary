# ✍️ Student Diary ✍️

A computer application for organising the 
academic life of students in Latvia. It 
provides a convenient way to store information 
about schedules, homework, deadlines, courses 
and lecturers throughout the academic year. 
The entire user interface is in Latvian!

## 🛠️ Features
- Weekly schedule with two-week rotation
- Homework tracker with priority levels
- Grades and statistics
- Deadline reminders
- Notes with search
- Teacher contact list

## ⚙️ Technologies
- C# and .NET 8
- Windows Forms
- SQLite
- library ScottPlot WinForms (charts)

## 💾 Download
Download the latest release StudentDiary.zip and run
StudentDiary.exe
